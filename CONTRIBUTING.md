# Contributing to agentation-vanilla

Thanks for your interest in contributing!

## How to contribute

1. Fork the repo
2. Create a branch (`git checkout -b my-feature`)
3. Make your changes
4. Test with `npm run demo` and open http://localhost:3333/demo
5. Commit and push to your fork
6. Open a pull request

## Guidelines

- Keep it simple :)
- Shadow DOM isolation matters — don't break out of it
- Please test your changes on the demo pages before submitting lol
- One PR per feature (ideally)

## Reporting bugs

Open an issue with steps to reproduce and what browser you're using.

## Questions?

Open an issue or reach out.
